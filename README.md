# lxqt-themes-devuan

This repository houses Devuan-styled forks of the LXQt Clearlooks theme. As of late January 2023, variants exist for DarkPurpy (Ascii), Cinnabar (Beowulf), and DeepSea (Chimaera), with Sapphire (Daedalus/testing) expected soon. Palettes for each theme have also been provided.

Below are previews of the themes running under LXQt 1.2.0. All previews show the Plastik Qt style and the Carbon Openbox theme, along with the respective LXQt themes and palettes.
